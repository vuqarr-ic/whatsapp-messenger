# Инструкция по установке и запуску

## Требования

- Node.js версии 16 или выше
- npm или yarn

## Установка

1. Откройте терминал в папке проекта:
```bash
cd whatsapp-messenger
```

2. Установите зависимости:
```bash
npm install
```

## Запуск

Запустите dev-сервер:
```bash
npm run dev
```

Приложение откроется по адресу `http://localhost:3000`

## Тестирование P2P функционала

Для тестирования обмена сообщениями:

1. Откройте приложение в двух разных вкладках браузера (или двух разных браузерах)
2. В первой вкладке:
   - Нажмите кнопку "+" для создания нового чата
   - Нажмите "Показать мой Peer ID" и скопируйте его
   - Введите имя (например, "Пользователь 1")
   - Введите Peer ID из второй вкладки
   - Создайте чат
3. Во второй вкладке:
   - Создайте чат с Peer ID из первой вкладки
4. Теперь вы можете обмениваться сообщениями между вкладками!

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Структура данных

Все данные хранятся в localStorage браузера:
- `whatsapp_chats` - список всех чатов
- `whatsapp_messages_<chatId>` - сообщения для каждого чата
- `whatsapp_peer_id` - ваш уникальный Peer ID

## Примечания

⚠️ **Текущая версия использует упрощенную P2P коммуникацию через localStorage для демонстрации.**

Для полноценной работы между разными устройствами требуется:
- Реализация WebRTC через библиотеку `simple-peer`
- Сигнальный сервер для обмена WebRTC сигналами
- STUN/TURN серверы для NAT traversal

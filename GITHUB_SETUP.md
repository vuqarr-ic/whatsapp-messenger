# Инструкция: Как загрузить проект на GitHub БЕЗ Git

## Шаг 1: Создайте репозиторий на GitHub

1. Зайдите на https://github.com и войдите (или зарегистрируйтесь)
2. Нажмите зелёную кнопку **"New"** (или плюсик в правом верхнем углу → "New repository")
3. Заполните:
   - **Repository name**: `stork-messenger` (или любое другое имя)
   - **Description**: `STORK - P2P мессенджер с WebRTC`
   - Выберите **Public** (чтобы Render и Netlify могли подключиться)
   - **НЕ** ставьте галочки на "Add a README file", "Add .gitignore" и т.п. (у вас уже есть файлы)
4. Нажмите **"Create repository"**

## Шаг 2: Загрузите файлы через веб-интерфейс

После создания репозитория GitHub покажет страницу с инструкциями. Выберите вариант **"uploading an existing file"**.

### Способ A: Загрузить всю папку через веб-интерфейс

1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите **ВСЕ файлы** из папки `C:\Users\masimov\whatsapp-messenger` в окно браузера
   - Или нажмите "choose your files" и выберите все файлы
3. Внизу страницы введите сообщение коммита: `Initial STORK version`
4. Нажмите **"Commit changes"**

### Способ B: Создать ZIP и загрузить

1. В проводнике Windows:
   - Откройте папку `C:\Users\masimov\whatsapp-messenger`
   - Выделите все файлы (Ctrl+A)
   - ПКМ → "Отправить" → "Сжатая ZIP-папка"
   - Назовите архив `stork-messenger.zip`

2. Распакуйте архив в пустую папку (например, `C:\temp\stork`)

3. На GitHub:
   - Нажмите "uploading an existing file"
   - Загрузите все файлы из распакованной папки

## Шаг 3: Проверьте

После загрузки обновите страницу репозитория. Вы должны увидеть все ваши файлы:
- `package.json`
- `src/`
- `server/`
- `README.md`
- и т.д.

Теперь у вас есть репозиторий на GitHub. Можете переходить к деплою на Render и Netlify.

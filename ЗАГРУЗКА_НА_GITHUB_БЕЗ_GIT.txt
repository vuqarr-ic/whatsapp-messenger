ИНСТРУКЦИЯ: Как загрузить проект на GitHub БЕЗ установки Git

═══════════════════════════════════════════════════════════════

ШАГ 1: Создайте репозиторий на GitHub
───────────────────────────────────────

1. Зайдите на https://github.com
2. Войдите (или зарегистрируйтесь)
3. Нажмите зелёную кнопку "New" (или плюсик → "New repository")
4. Заполните:
   - Repository name: stork-messenger
   - Description: STORK - P2P мессенджер
   - Выберите: Public
   - НЕ ставьте галочки на "Add README", "Add .gitignore" и т.п.
5. Нажмите "Create repository"


ШАГ 2: Загрузите файлы через браузер
───────────────────────────────────────

После создания репозитория GitHub покажет страницу с инструкциями.

1. На этой странице нажмите ссылку "uploading an existing file"
   (или просто перейдите на вкладку "Code" → "upload files")

2. Откройте папку проекта в проводнике Windows:
   C:\Users\masimov\whatsapp-messenger

3. Выделите ВСЕ файлы и папки (Ctrl+A):
   - package.json
   - package-lock.json
   - vite.config.js
   - index.html
   - .gitignore
   - README.md
   - DEPLOY.md
   - INSTALL.md
   - ПОЛНАЯ_ИНСТРУКЦИЯ.md
   - USER_GUIDE.md
   - CHECKLIST.md
   - GITHUB_SETUP.md
   - ЗАГРУЗКА_НА_GITHUB_БЕЗ_GIT.txt
   - папка src/
   - папка server/
   - папка public/
   - и все остальные файлы

4. Перетащите все файлы в окно браузера на GitHub
   (или нажмите "choose your files" и выберите все файлы)

5. Внизу страницы в поле "Commit changes" введите:
   Initial STORK version

6. Нажмите зелёную кнопку "Commit changes"


ШАГ 3: Проверьте
──────────────────

1. Обновите страницу репозитория (F5)
2. Вы должны увидеть все ваши файлы:
   - package.json
   - src/
   - server/
   - README.md
   - и т.д.

3. Если всё на месте - готово! ✅
